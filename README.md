# LiraRate
A project to display the price of the Lebanese pound on the black market made in Python

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the libraries.

```bash
pip install -r .\requirements.txt
```
## Usage

```bash
python python.py
```
